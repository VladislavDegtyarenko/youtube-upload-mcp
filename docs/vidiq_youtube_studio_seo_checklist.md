# SEO оптимизация видео при загрузке в YouTube Studio

Выжимка советов по материалам блога vidIQ. Фокус - на действиях, которые можно сделать именно на этапе загрузки ролика в YouTube Studio.

## Главная мысль

SEO при загрузке - это не просто "накидать тегов". Это упаковка ролика:

- тема
- ключевой запрос
- заголовок
- обложка
- первые строки описания
- главы
- субтитры
- плейлист
- эндскрины
- последующая итерация после публикации

Метаданные помогают YouTube понять видео, но дальше всё решают клики и удержание. Если заголовок и обложка не дают CTR, а начало ролика не удерживает зрителя, одни теги не спасут.

Источник: https://vidiq.com/blog/post/youtube-seo-for-brands/

---

## Чеклист на этапе загрузки

## 1. До загрузки выбери 1 главный keyword

Не начинай с названия "как красиво звучит". Сначала найди фразу, которую реально ищут:

- YouTube autocomplete
- похожие запросы
- конкуренты
- vidIQ keyword score
- long-tail варианты

vidIQ советует искать не просто популярный запрос, а opportunity keyword: достаточный search volume плюс конкуренция, которую твой канал способен пробить.

Источник: https://vidiq.com/ru/blog/post/how-to-rank-number-one-youtube-keyword-research/

### Примеры для coding tutorial

```text
YouTube thumbnail downloader JavaScript
Build a YouTube thumbnail downloader with HTML CSS JavaScript
Beginner JavaScript project YouTube thumbnail downloader
```

---

## 2. Назови файл видео ключевым запросом

Перед upload переименуй файл не в:

```text
final_v3_export.mp4
```

А в понятное SEO-имя:

```text
youtube-thumbnail-downloader-javascript-tutorial.mp4
```

vidIQ включает filename в on-page SEO: keyword в title, first line description и filename перед загрузкой.

Источник: https://vidiq.com/blog/post/youtube-seo/

---

## 3. Сначала загружай как Private или Unlisted

Во время загрузки лучше не ставить Public сразу. Начни с Private или Unlisted, чтобы спокойно проверить:

- title
- description
- thumbnail
- chapters
- processing quality
- monetization
- compliance-настройки
- end screen
- cards

Источник: https://vidiq.com/ru/blog/post/how-to-upload-videos-to-youtube-creator-studio/

---

## 4. Title: ключевик ближе к началу, но не сухой SEO-заголовок

Формула:

```text
keyword + понятная польза + причина кликнуть
```

vidIQ советует держать title примерно в 60-70 символов, ставить главный keyword ближе к началу, избегать кликбейта и делать title в связке с thumbnail.

Источник: https://vidiq.com/blog/post/youtube-video-title/

### Примеры

Плохо:

```text
My New JavaScript Project
```

Нормально:

```text
JavaScript YouTube Thumbnail Downloader - Beginner Project
```

Сильнее:

```text
Build a YouTube Thumbnail Downloader with JavaScript
```

Для первого пуша можно сделать чуть более curiosity-driven title, а потом через 48-72 часа перевернуть его в более SEO-вариант, если видео проседает. vidIQ называет это Title Flip Framework: сначала title под Browse/Suggested, потом title под Search.

Источник: https://vidiq.com/blog/post/YouTube-Title-Flip-SEO-Framework/

---

## 5. Thumbnail должен усиливать title, а не повторять его

Обложка - часть SEO, потому что она влияет на CTR.

vidIQ повторяет:

- один фокус
- высокий контраст
- читаемость на телефоне
- минимум мусора
- 2-3 сильных визуальных элемента

Технически:

- 1280x720
- 16:9
- до 2 MB
- JPG, PNG или WebP

Источник: https://vidiq.com/blog/post/youtube-seo/

### Пример для coding tutorial

Title:

```text
Build a YouTube Thumbnail Downloader with JavaScript
```

Thumbnail text:

```text
GET ANY THUMBNAIL
```

или:

```text
JS PROJECT
```

Visual idea:

```text
YouTube URL -> preview cards -> download buttons
```

---

## 6. Первые 100-150 символов описания - самые важные

Первые строки описания работают как search snippet. Не начинай описание с "Hey guys" или ссылок. Сразу дай hook плюс главный keyword.

Источник: https://vidiq.com/ru/blog/post/youtube-video-descriptions/

### Шаблон первой строки

```text
Build a YouTube Thumbnail Downloader with HTML, CSS, and JavaScript. In this beginner project, we'll parse a YouTube URL, extract the video ID, and render downloadable thumbnails.
```

После этого можно добавить:

- что зритель научится делать
- GitHub link
- live demo
- related video
- subscribe CTA
- chapters
- 2-3 hashtags

---

## 7. Не делай описание огромной простынёй

Поле description поддерживает до 5000 символов, но обычно не нужно забивать его полностью. Практичный ориентир для tutorial-видео:

```text
700-1500 символов
```

Этого обычно хватает для:

- SEO-контекста
- структуры
- ссылок
- chapters
- hashtags
- CTA

Источник: https://vidiq.com/ru/blog/post/youtube-video-descriptions/

---

## 8. Chapters добавляй сразу при загрузке

Для видео длиннее 8-10 минут добавляй таймкоды. Chapters помогают зрителю навигироваться, дают YouTube дополнительный контекст и могут появляться как key moments в поиске.

Главное:

- начинать с `0:00`
- давать понятные названия секций
- не делать названия слишком общими

Источник: https://vidiq.com/ru/blog/post/youtube-video-descriptions/

### Пример

```text
0:00 - Demo
0:35 - Project setup
2:10 - HTML structure
5:40 - Extracting YouTube video ID
9:20 - Rendering thumbnail options
14:30 - Download buttons
18:00 - Final result
```

---

## 9. Hashtags: 2-3, не больше по смыслу

Добавляй 2-3 релевантных hashtags в описание. Лучше смешивать широкий и нишевый hashtag.

Источник: https://vidiq.com/blog/post/youtube-seo/

### Пример

```text
#JavaScript #WebDevelopment #CodingTutorial
```

Не стоит лепить 10-15 хэштегов. Это выглядит спамно и размывает сигнал.

---

## 10. Tags заполняй, но не поклоняйся им

Tags сейчас слабее, чем title, description, thumbnail и retention. Но они полезны для уточнения темы, misspellings, брендов, технологий и неоднозначных слов.

vidIQ пишет, что tags - minor ranking signal, но помогают YouTube disambiguate topic.

Источник: https://vidiq.com/blog/post/youtube-seo/

### Пример для coding tutorial

```text
javascript tutorial, html css javascript project, youtube thumbnail downloader, beginner javascript project, web development tutorial, vanilla javascript
```

---

## 11. Добавь видео в правильный playlist

Playlist - это не просто сортировка. Он помогает discoverability, потому что люди могут искать плейлисты, а ещё это увеличивает binge-watching.

Источник: https://vidiq.com/blog/post/youtube-seo/

### Примеры плейлистов

```text
JavaScript Projects for Beginners
HTML CSS JavaScript Tutorials
Frontend Projects
```

---

## 12. Проверь captions/subtitles

Даже если YouTube сам генерирует auto-captions, проверь имена, бренды, технологии и термины.

Для coding-контента авто-субтитры часто ошибаются в словах вроде:

```text
query param
localStorage
createElement
thumbnail
fetch
Vite
React
Next.js
```

Источник: https://vidiq.com/blog/post/youtube-seo/

---

## 13. End screen и cards - сразу веди на следующий ролик

В момент загрузки добавь end screen. Лучше не 5 вариантов, а 1-2 логичных следующих видео или playlist.

Также можно использовать cards и pinned comment, чтобы направлять зрителя к следующему лучшему видео на канале.

Источник: https://vidiq.com/blog/post/youtube-seo/

### Пример

End screen:

```text
Build a Weather App with JavaScript
```

или:

```text
More JavaScript Projects
```

Pinned comment:

```text
Source code + live demo here: ...
```

---

## 14. Schedule по активности аудитории

Если ролик готов, не всегда нужно жать Publish сразу. Лучше посмотреть в YouTube Analytics, когда аудитория активнее, и использовать Schedule.

Источник: https://vidiq.com/ru/blog/post/how-to-upload-videos-to-youtube-creator-studio/

---

## 15. После публикации пересмотри SEO через 48-72 часа, 7 дней и 28 дней

Это уже не upload form, но это часть SEO-процесса.

После первых 48-72 часов смотри:

- impressions
- CTR
- watch time
- audience retention
- traffic source
- search terms

Если impressions есть, но CTR низкий - меняй title или thumbnail.

Если impressions низкие - усиливай keyword, description, chapters и related packaging.

Через 7 и 28 дней проверь Reach tab: какие поисковые запросы дали impressions, что можно внести в title, description или chapters.

Источник: https://vidiq.com/blog/post/YouTube-Title-Flip-SEO-Framework/

# Короткая формула

```text
Keyword research -> filename -> title -> thumbnail -> first lines of description -> chapters -> subtitles -> playlist -> end screen -> post-upload iteration
```

Итоговая идея vidIQ:

```text
Не SEO ради SEO, а понятный сигнал для алгоритма плюс сильная упаковка для человека.
```

Title и description помогают YouTube понять видео. Thumbnail и title заставляют кликнуть. Retention решает, будет ли YouTube дальше расширять показы.
